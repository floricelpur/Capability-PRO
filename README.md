## ğŸ“Š Capability Indices Explained

- **Cp**: Process capability potential (if perfectly centered)
- **Cpk**: Process capability considering centering (actual performance)
- **Pp**: Process performance potential (long-term)
- **Ppk**: Process performance considering centering (long-term)
- **Cpu/Cpl**: Capability relative to upper or lower specification limits
- **K**: Process shift indicator showing how far mean is from target
